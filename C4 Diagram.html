üèóÔ∏è C4 Context Diagram Example

This Mermaid block renders a simple C4 Context diagram showing a user, a system, and an external dependency. You can expand this into Containers, Components, or Dynamic views as needed.

C4Context
  title System Context diagram for Internet Banking System
  Person(customer, "Customer", "A customer of the bank, with personal accounts")
  System(banking_system, "Internet Banking System", "Allows customers to view information and make payments")
  System_Ext(mail_system, "E-mail system", "The internal Microsoft Exchange system")
  System_Ext(mainframe, "Mainframe Banking System", "Stores all customer information")

  Rel(customer, banking_system, "Uses")
  Rel_Back(banking_system, mail_system, "Sends e-mails using")
  Rel(banking_system, mainframe, "Uses for core banking data")
üõ†Ô∏è How to Customise
  • People: Use Person(id, "Name", "Description")
  • Systems: Use System or System_Ext for external ones
  • Relationships: Use Rel or Rel_Back to connect elements
  • Other levels: Try C4Container, C4Component, or C4Dynamic for deeper detail
