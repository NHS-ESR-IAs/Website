    This Mermaid block renders a Git Graph showing a simple repo flow with feature branches, a hotfix, and tagged releases. You can customise branch names, commit order, and tags to reflect your own workflow.

    gitGraph
  commit id: "init"
  commit tag: "v1.0.0"
  branch feature/auth
  checkout feature/auth
  commit id: "auth-1"
  commit id: "auth-2"
  checkout main
  merge feature/auth tag: "v1.1.0"

  branch hotfix/login
  checkout hotfix/login
  commit id: "hotfix-1"
  checkout main
  merge hotfix/login tag: "v1.1.1"

  branch feature/ui
  checkout feature/ui
  commit id: "ui-1"
  commit id: "ui-2"
  checkout main
  merge feature/ui tag: "v1.2.0"

    • Branch strategy: Use branch, checkout, and merge to model your flow

    • Tags: Append tag: "vX.Y.Z" to mark releases
    • Layout: You can add an options { } block under gitGraph to tweak spacing and node radius
