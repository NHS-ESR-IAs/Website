This Mermaid block renders a class diagram showing how a microsite might relate to plugins and snippets. You can customise the classes, methods, and relationships to suit your own architecture.

classDiagram
  class Microsite {
    +showPage()
    +copySnippet()
  }

  class Plugin {
    +activate()
    +deactivate()
  }

  class Snippet {
    +language
    +copyToClipboard()
  }

  Microsite --> Plugin : loads
  Microsite --> Snippet : contains
  Plugin <|-- CopyButton
  Plugin <|-- Mermaid
  Plugin <|-- PlantUML
